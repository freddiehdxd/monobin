# AGENTS.md

A Monobin site (Go single-binary SSR + islands).

## Commands
- `go run . dev` — dev server (reads `app/` from disk; run from the repo root)
- `cd islands && npm install && npm run build` — build client assets into `app/assets/`
- `go build -o app-bin .` — single binary (NOT `-o app`; `app/` is a directory)
- `monobin`-style CLI on the built binary: `serve` · `build [dir]` · `routes [--json]` · `check [--json]`

## Non-obvious patterns
- **Islands must be registered** in `islands/src/entry.js` (the `islands = {}` map) or they won't mount. `check` flags this.
- **Tailwind `@source`**: classes used only in `app/**.html` are purged unless `islands/src/style.css` keeps `@source "../../app"`.
- Add a page: drop a template in `app/routes/` (`about.html` → `/about`, `blog/[slug].html` → `/blog/:slug`).
- Server data (→ template `.Data`): in `main.go`, `app.Loader("/path", func(c *framework.Ctx) (any, error) { ... })`; dynamic routes also need `app.StaticPaths(...)` to be pre-rendered by `build`.

## Boundaries
- Don't edit `app/assets/*` — generated by Vite.
- Run `check` before committing.
